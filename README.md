# Loss-of-function (LoF) variant annotations
Repository of codes and data to annotate loss-of-function variants from whole-exome sequencing data (WES) in the UK Biobank <br>
The order of execution of the scripts is in the folder `annotations`. <br> File paths and more references are in `Data` folder <br>
